# Online-Code-Compiler

This is a very basic code editor that uses API from [qCompiler](https://www.npmjs.com/package/qcompiler) package. The editor uses APIs from [CodeMirror](https://codemirror.net/).

## Supported Languages

- C
- CPP
- JAVA
- PYTHON
